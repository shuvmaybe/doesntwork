Stop using 'Don't work' or 'Doesn't work'

Instead save yours, and everybody else's time.

Grown tired of coming to a chatroom just to see a person saying they followed all the steps for something and suddenly it doesn't work without any additional information.

Man, what do I do with this information? How do you expect us to help you without knowing what the problem is?

What you should learn is whenever you have an issue and need help, make sure to format your message with this information:

  • What is not happening: Clearly state what is not happening as expected. For example: "I was trying to get this value."
  • What should happen: Describe the expected outcome. For instance: "It should get the value."
  • Provide error messages & additional information: Include any error messages encountered and relevant context. For instance: "Value does not exist."

Here is how time wasteful using "Doesn't work" is:

[10:15 AM] [Joe Biden] Hello guys, I installed this theme and it doesn't work.
[10:17 AM] [Donald Trump] What is not working specifically?
[10:18 AM] [Joe Biden] Do I need a different plugin for the theme?
[10:19 AM] [Donald Trump] What is not working with the theme?
[10:21 AM] [Joe Biden] It is all white, there is supposed to be a black background.
[10:22 AM] [Donald Trump] The background links for themes have expired. Replace your background link with this one. [solution]
[10:23 AM] [Joe Biden] Works! Thanks!

It took Joe 8 minutes to get his problem fixed, just because he went straight to 'doesn't work' instead of explaining the problem.

What Joe should of have done instead of immediately whining is to explain the issue he is encountering and what he expected to happen; in this case his theme was supposed to have a black background, instead it was white.

[10:15 AM] [Joe Biden] What's up guys, I was installing this theme and it is completely white instead of having a dark background. How can I fix this?
[10:17 AM] [Donald Trump] The background links for themes have expired. Replace your background link with this one. [solution]
[10:18 AM] [Joe Biden] Works! Thanks!

Joe's issue was now fixed in 3 minutes! 26% Faster than with just saying 'doesn't work' every couple of messages.

When 'Doesn't work' is acceptable:

While the importance of explaining your issue right away is emphasized and resolved, there are times when using "doesn't work" is appropriate:

For example, when seeking solutions known to have similar error messages or identical situations, such as:

[10:30 AM] [Barack Obama] Hello, my code seems to error due to *error 1*, my code is: [code]
[10:32 AM] [Donald Trump] It seems like you are using the function improperly. [solution]
[10:33 AM] [Barack Obama] Doesn't work.
[10:34 AM] [Donald Trump] Ah, I see the issue now. [solution]
[10:34 AM] [Barack Obama] Now it works! Thanks.

Here Obama, even with using 'Doesn't work' resolved his issue fairly quickly. Since Donald already knew the error he presumed that the error from the first message stayed.

Obama, even with using the forbidden term, fixed his issues because of correct usage. Be more like Obama.
